Поддержка VOIP для Homey.

Настройте данные вашей SIP учетной записи через настройки приложения в Homey, чтобы домен, имя пользователя, ID аутентификации, пароль, область и порты можно было настроить по вашему усмотрению.

При желании можно указать STUN-сервер для определения публичного IP-адреса и портов. Это может помочь с NAT-проблемами при входящих SIP и RTP.

В настройках можно выбрать кодек. По умолчанию приложение автоматически пытается использовать лучшее качество (G722 > PCMA > PCMU) и переходит на более низкое качество, если это требуется SIP-сервером.

Аудиофайлы в формате WAV или MP3 автоматически конвертируются в подходящий моно 8kHz формат. Разумеется, чем больше аудиофайл, тем дольше длится запуск SIP-звонка.

Важно иметь готовые объявления в правильном формате на звуковой доске или в другом месте.
